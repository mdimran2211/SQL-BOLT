# SQL Exercise 3 – Filtering by Name and Director

## Objective
Practice SQL filtering using `WHERE`, `NOT`, and `LIKE` operators on the `movies` table.

---

## Queries

1️⃣ Find all the Toy Story movies  
```sql
SELECT * FROM movies
WHERE title LIKE 'Toy Story%';
2️⃣ Find all the movies directed by John Lasseter

SELECT * FROM movies
WHERE director = 'John Lasseter';

3️⃣ Find all the movies (and director) not directed by John Lasseter

SELECT title, director FROM movies
WHERE director != 'John Lasseter';

4️⃣ Find all the WALL-* movies

SELECT * FROM movies
WHERE title LIKE 'WALL-%';
